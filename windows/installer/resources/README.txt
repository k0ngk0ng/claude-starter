Claude Code for Windows
=======================

Thank you for installing Claude Code!

Getting Started:
1. Launch "Claude Code" from your Desktop or Start Menu
2. Configure your ANTHROPIC_API_KEY in the launcher
3. Click "Launch Claude Code" to start

Configuration:
- All settings are saved in: %APPDATA%\ClaudeCode\config.json
- You can modify settings at any time through the launcher

Available Settings:
- ANTHROPIC_API_KEY: Your Anthropic API key
- ANTHROPIC_MODEL: Specify a particular Claude model
- ANTHROPIC_BASE_URL: Custom API endpoint (for proxies)
- ANTHROPIC_TIMEOUT: Request timeout in milliseconds
- HTTP_PROXY / HTTPS_PROXY: Proxy settings
- --dangerously-skip-permissions: Skip permission prompts (use with caution!)

Bundled Components:
- Node.js (LTS version)
- Git for Windows
- Claude Code CLI (latest)

For more information, visit: https://claude.ai

Documentation: https://docs.anthropic.com/en/docs/claude-code
